# customer-churn-analysis
Data Science Internship Project - Telco Customer Churn Analysis

## About This Project
As part of my Data Science internship at Innovexa Catalyst, I worked on 
analyzing a Telco company's customer data to understand why customers leave 
and build a model to predict churn.

## What I Did
- Loaded and explored the dataset to understand its structure
- Cleaned the data by handling missing values and removing inconsistencies
- Performed exploratory analysis to find patterns and trends
- Created visualizations to communicate insights clearly
- Built machine learning models to predict whether a customer will churn

## What I Found
- Customers on month-to-month contracts are much more likely to leave
- New customers (low tenure) with high monthly bills churn the most
- Fiber optic internet users showed higher churn compared to DSL users
- Electronic check payment users had the highest churn rate
- Gender had almost no effect on churn

## Models Built
I tried two models — Logistic Regression and Random Forest. 
Random Forest performed better with around 80%+ accuracy.

## Tools & Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

## Dataset
Telco Customer Churn dataset with 7043 customers and 21 features including 
contract type, payment method, internet service, monthly charges and more.
